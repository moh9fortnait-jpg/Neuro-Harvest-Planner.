# 12. Neuro-Harvest Planner

## Concept
A digital assistant for rural farming communities. It helps farmers estimate their potential crop yield (Wheat, Barley, or Potato) based on the square footage of their land and regional agricultural standards.

## Innovation
This project bridges the gap between **Traditional Agriculture and Data**. It uses a "Mapping Logic" (Dictionary) to store yield rates per hectare, allowing for instant projections that help in logistics and storage planning.

## How to Run
1. Run: `python main.py`
2. Enter the land area in square meters.
3. Select the crop type to receive an estimated harvest weight in tons.
